# 에이전트 자기 평가 템플릿

이 템플릿은 모든 에이전트가 작업 완료 후 자기 평가 시 사용하는 공통 형식입니다.

## 📊 작업 결과 자기 평가

### 평가 기준 (총 100점)

#### 1. 완성도 (30점)

- 요구사항을 모두 충족했는가?
- 빠진 부분은 없는가?
- 엣지 케이스를 고려했는가?

#### 2. 품질 (30점)

- 베스트 프랙티스를 따랐는가?
- 규칙과 컨벤션을 준수했는가?
- 재사용 가능하고 유지보수하기 쉬운가?

#### 3. 명확성 (20점)

- 의도가 명확하게 전달되는가?
- 이해하기 쉬운가?
- 문서화가 적절한가?

#### 4. 일관성 (20점)

- 프로젝트 규칙과 일치하는가?
- 기존 산출물과 스타일이 통일되는가?
- 다른 에이전트의 작업과 연결이 매끄러운가?

### 평가 보고서 형식

```
📊 [에이전트명] 작업 결과 자기 평가

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
종합 점수: XX/100 (등급)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 항목별 점수

1. 완성도: XX/30 ⭐⭐⭐⭐⭐

   - [구체적인 근거 1]
   - [구체적인 근거 2]
   ⚠️ 개선점:
   - [개선이 필요한 점 1] (-X점)

   ✅ 근거:
   - [구체적인 근거]
   ⚠️ 개선점:
   - [개선이 필요한 점] (-X점)

3. 명확성: XX/20 ⭐⭐⭐⭐⭐
   ✅ 근거:
   - [구체적인 근거]
   ⚠️ 개선점:
   - [개선이 필요한 점] (-X점)

4. 일관성: XX/20 ⭐⭐⭐
   ✅ 근거:
   - [구체적인 근거]
   ⚠️ 개선점:
   - [개선이 필요한 점] (-X점)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 점수 기준

- **90-100점 (A)**: 탁월함, 즉시 다음 단계 진행 권장
- **80-89점 (B)**: 우수함, 사소한 개선 후 진행 가능
- **70-79점 (C)**: 보통, 개선 권장하나 진행 가능
- **60-69점 (D)**: 미흡, 주요 개선 필수
- **60점 미만 (F)**: 불합격, 재작업 필요

### 평가 투명성 원칙

1. **객관적 근거**: 모든 점수에는 구체적인 근거 제시
2. **구체적 개선점**: 파일명, 위치, 내용까지 명시
3. **우선순위 표시**: 높음/중간/낮음으로 구분
4. **감점 이유 설명**: 왜 감점했는지 명확히

## ⚠️ 불명확한 정보 처리 원칙 (공통)

### 절대 규칙

- ❌ **추측하지 않습니다**
- ❌ **가정하고 진행하지 않습니다**
- ✅ **명확하지 않으면 반드시 질문합니다**
- ✅ **구체적인 선택지를 제시합니다**

   - 옵션 A: [구체적 설명]

명확한 답변을 주시면 정확한 작업을 진행하겠습니다.
```
